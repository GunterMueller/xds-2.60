Проект:		$/XDSmisc
Назначение: 	рабочий проект для различных частей XDS. Дополняет остальные рабочие проекты продуктов.
Статус:		Используется во время выпуска релизов XDS
Хранить до:     Пока идет работа над продуктами

Содержание
----------
  BURG - исходники BURG'а, используемого для XDS-компайлера
  DOC  - исходники пользовательской документации всех продуктов, структурированные в виде, удобном для сборки вместе с исходниками конвертора
  readme - readme-файлы, укладываемые в дистрибутивы продуктов
  Samples - программы-примеры, входящие в дистрибутив XDS
